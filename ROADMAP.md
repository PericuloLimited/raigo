# RAIGO Roadmap

This document outlines the high-level roadmap for the RAIGO project. It is intended to provide transparency into our current priorities and future direction. Please note that this roadmap is subject to change based on community feedback and emerging requirements.

## Current Focus (Q2 2026)

*   **Launch & Adoption:** Officially launch RAIGO v1.0 and drive adoption within the OpenClaw community and target sectors (Defence, Healthcare).
*   **Compiler Enhancements:** Improve the stability and performance of the RAIGO compiler.
*   **Web Generator MVP:** Build and launch the Lovable-powered web interface for easy `.raigo` file generation.
*   **Expanded Targets:** Add support for more AI platforms and tools based on community requests.

## Near Term (Q3 2026)

*   **VS Code Extension:** Develop a VS Code extension for `.raigo` files with syntax highlighting, linting, and live compilation preview.
*   **CI/CD Integrations:** Create GitHub Actions and GitLab CI templates for automated `.raigo` policy validation in pipelines.
*   **Policy Registry:** Establish a public registry for sharing and discovering community-contributed `.raigo` policies.
*   **Advanced Policy Features:** Introduce support for policy inheritance, modularization, and dynamic variables.

## Long Term (Q4 2026 and Beyond)

*   **Runtime Interceptor Engine (WAF for AI):** Develop a standalone runtime engine that acts as a proxy between applications and AI models, enforcing `.raigo` policies in real-time.
*   **Enterprise Control Plane:** Launch the commercial Periculo Enterprise Control Plane for centralized policy management, audit logging, and fleet-wide synchronization.
*   **Microsoft Copilot Studio Integration:** Build a deep, native integration with Microsoft Copilot Studio, leveraging the runtime interceptor architecture.
*   **Formal Standardization:** Work towards submitting the `.raigo` specification to recognized standards bodies.

## How to Contribute to the Roadmap

We welcome feedback and suggestions for the roadmap! If you have ideas for new features or improvements, please:

1.  Check the [issue tracker](https://github.com/PericuloLimited/raigo/issues) to see if it has already been proposed.
2.  If not, open a new issue describing your idea and its use case.
3.  Participate in community discussions to help prioritize features.
