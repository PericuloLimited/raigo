# RAIGO Adopters

This document lists organizations that have adopted RAIGO in production. If your organization uses RAIGO, please consider submitting a pull request to add yourself to this list. This helps demonstrate the value of the project and encourages others to adopt it.

## Production Adopters

| Organization | Industry | Use Case |
|---|---|---|
| [Periculo Security](https://periculo.co.uk) | Cybersecurity | Internal AI agent governance across all client-facing AI tools |

## How to Add Your Organization

If you are using RAIGO in production, please submit a pull request to add your organization to this list. Include the following information:

*   **Organization name** with a link to your website.
*   **Industry** (e.g., Healthcare, Defence, Finance, Technology).
*   **Use case** — a brief description of how you are using RAIGO.

We welcome all organizations, from startups to large enterprises, to share their RAIGO use cases.
